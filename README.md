## 🚀 Sobre o Projeto

Este portfólio foi construído do zero com foco em **Performance, Acessibilidade e Design Moderno**. A interface utiliza conceitos de *Glassmorphism* (Efeito de Vidro), paleta de cores escura (*Dark Mode*) com contrastes em neon (vermelho primário) e animações suaves de rolagem para proporcionar uma experiência de usuário (UX) premium.

### 🌟 Principais Seções:
- **Hero/Header:** Apresentação inicial marcante com efeito visual *full-bleed* e destaques das minhas certificações e objetivo de transição de carreira.
- **Sobre Mim (Dashboard):** Uma linha do tempo interativa e estruturada com CSS Grid, dividindo de forma clara minha **Experiência Profissional** (com destaque em Liderança), **Formação Acadêmica** e **Habilidades (Hard/Soft Skills)** através de um componente Accordion elegante.
- **Projetos:** Um *Solo Card* interativo, com efeitos 3D (*Hover Glow*) destacando meu repositório no GitHub para visualização rápida dos meus códigos e contribuições Open Source.
- **Contato:** Área de networking direto, incluindo botão com API inteligente de "Copiar" (Clipboard) para e-mail/celular e links diretos para LinkedIn e Instagram.

## 🛠️ Tecnologias Utilizadas

O projeto foi inteiramente construído com tecnologias web nativas para garantir máxima velocidade e leveza na renderização (sem dependência pesada de frameworks):

- **HTML5:** Estrutura semântica e acessível.
- **CSS3 (Vanilla):** 
  - Layouts complexos utilizando **CSS Grid e Flexbox**.
  - Efeitos premium de transparência e desfoque (`backdrop-filter`).
  - Animações CSS Keyframes (`@keyframes pulseDot`).
  - Design Responsivo para Mobile e Desktop (`@media queries`).
- **JavaScript (Vanilla):**
  - **Intersection Observer API:** Usada para disparar animações apenas quando o usuário rola até as seções específicas da página.
  - **Clipboard API:** Usada para a funcionalidade de clique em "Copiar" nos componentes de contato.
  - Funções de alternância dinámica de estados (Acordeão de habilidades e Navbar).
- **Google Fonts:** Tipografia moderna utilizando a fonte *Outfit*.
- **FontAwesome:** Biblioteca de ícones (SVG).

## 📱 Responsividade

O portfólio é **100% responsivo**. Cartões que aparecem lado a lado em monitores tradicionais (*desktop*) se ajustam de forma harmoniosa em um empilhamento vertical suave nos dispositivos móveis (*smartphones e tablets*), garantindo que nenhum texto seja cortado e a legibilidade seja sempre privilegiada.

## ⚙️ Como executar localmente

Caso você deseje rodar o site localmente no seu computador:

1. Faça o clone deste repositório:
   ```bash
   git clone https://github.com/deborabasilio/portfolio.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd portfolio
   ```
3. Abra o arquivo `index.html` em qualquer navegador web moderno:
   - Duplo clique no arquivo `index.html`
   - *Se preferir usar o VS Code: Instale a extensão "Live Server" e inicie por lá.*
